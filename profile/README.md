## // WHO WE ARE

**Cyberus Security Project** is an independent, open-source security research project based in Athens, Greece — built by **Ailina S.** & **Michael C.**

We're not a company. We're researchers, builders, and learners who work on problems we find interesting: cloud security, security analysis, AI in cybersecurity, and security tool development.

Everything we build is open and lives here on GitHub. Our goal is to build useful things, document our process honestly, and share what we figure out — because **good security knowledge should be accessible to everyone.**

Whether you're a fellow researcher, a hiring manager, or just curious — everything here is open to explore.
