## Reporting a Vulnerability

If you discover a security vulnerability in any Cyberus repository, **please do not open a public issue.**

Report it privately via:
- **Email**: security@cyberus.gr
- **GitHub Private Advisory**: Use the "Report a vulnerability" button on the affected repo

We aim to respond within **72 hours** and will coordinate disclosure with you.

## Disclosure Policy

We follow **responsible disclosure**. Once a fix is released, we'll credit the reporter (unless anonymity is requested).
